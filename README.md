# AWS S3 com AWS SDK
Projeto para acesso e  manipulação de arquivos na AWS S3

# Configuração do projeto!

Criar as variáveis de ambiente que são **accessKey** e **secretKey**, que são as credenciais criadas em sua conta Amazon AWS.
